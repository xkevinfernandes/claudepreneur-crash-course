# 5 Agent Workflow Templates

> Copy any prompt below. Paste into Claude Code. Replace [BRACKETS]. Hit enter.
> Each one spawns 3 agents that work simultaneously.

---

## 1. Niche Deep Dive

Research your entire niche in parallel — pain points, competitors, and content gaps.

```
I need you to research my niche. Spawn 3 agents in parallel:

Agent 1 — Pain Points: Use WebSearch to find what my ideal clients are searching for, complaining about, and asking in Reddit, forums, and communities. What do they wish existed? What keeps them up at night? Save to ./research-pain-points.md

Agent 2 — Competitor Landscape: Use WebSearch to find who's operating in my niche, what they charge, how they position themselves, and where they're weak. Find 4-5 competitors. Save to ./research-competitors.md

Agent 3 — Content Gaps: Use WebSearch to find what content exists in my niche, what topics are underserved, and what angles nobody is taking. Check YouTube, Instagram, blogs. Save to ./research-content-gaps.md

My niche: [YOUR NICHE]
My ideal client: [YOUR IDEAL CLIENT]
```

## 2. Content Blitz

One topic, three content pieces, all at the same time.

```
I need a content blitz on this topic: [YOUR TOPIC]

Spawn 3 agents in parallel:

Agent 1 — Instagram Post: Write an Instagram caption about this topic. Hook, value, CTA. In my voice (read my CLAUDE.md). Save to ./content-instagram.md

Agent 2 — Email Newsletter: Write a short email newsletter about this topic. Subject line, opening hook, 3 key points, CTA. In my voice. Save to ./content-email.md

Agent 3 — Video Script: Write a 60-second video script about this topic. Opening hook (3 seconds), main content, closing CTA. Conversational, not scripted-sounding. Save to ./content-video-script.md
```

## 3. Client Engagement Pack

Welcome message, check-in template, and testimonial request — all created at once.

```
I need a client engagement pack. Spawn 3 agents in parallel:

Agent 1 — Welcome Message: Write a warm, professional welcome message for new clients. Set expectations, show them what to expect, make them feel confident they made the right choice. Save to ./client-welcome.md

Agent 2 — Weekly Check-In: Create a weekly check-in template I can send to clients. Ask about progress, celebrate wins, identify blockers, suggest next steps. Save to ./client-checkin.md

Agent 3 — Testimonial Request: Write a natural, non-awkward message asking a client for a testimonial. Make it easy for them to respond. Include 3 specific prompt questions. Save to ./client-testimonial-request.md

My business: [WHAT YOU DO]
My clients: [WHO YOU SERVE]
```

## 4. Workshop Engine

Plan an entire workshop — outline, promo content, and follow-up emails.

```
I'm running a workshop on: [WORKSHOP TOPIC]

Spawn 3 agents in parallel:

Agent 1 — Workshop Outline: Create a 60-minute workshop structure. Opening hook, 3 main teaching sections, interactive elements, closing CTA. Include timing for each section. Save to ./workshop-outline.md

Agent 2 — Promo Content: Write 3 social media posts promoting the workshop. Different angles — pain, desire, credibility. Include hooks and CTAs. Save to ./workshop-promo.md

Agent 3 — Follow-Up Emails: Write 2 emails — one for attendees (recap + next step CTA) and one for no-shows (what they missed + replay access). Save to ./workshop-followup.md
```

## 5. Weekly CEO Brief

Market trends, content ideas, and an action plan — your weekly intelligence report.

```
Give me my weekly CEO brief. Spawn 3 agents in parallel:

Agent 1 — Market Trends: Use WebSearch to find what's trending in my niche this week. New conversations, viral posts, emerging topics. What should I be aware of? Save to ./weekly-trends.md

Agent 2 — Content Ideas: Based on what's happening in my niche (use WebSearch), generate 7 content ideas for next week. Each one with a hook, angle, and format suggestion. Save to ./weekly-content-ideas.md

Agent 3 — Action Plan: Review my current priorities and suggest my top 3 actions for next week. What will move the needle most? Save to ./weekly-action-plan.md

My niche: [YOUR NICHE]
My current priorities: [WHAT YOU'RE FOCUSED ON]
```
