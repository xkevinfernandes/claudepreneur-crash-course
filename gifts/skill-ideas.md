## 1. Session Prep

**Name:** `my-session-prep`
**What it does:** Creates a prep sheet before a client session.

```markdown
---
description: "Create a prep sheet for a client session. Context, goals, questions to ask."
---

# Session Prep

Read CLAUDE.md and brand files for context.

Ask me: Who is the client and what are we working on?

Then create a prep sheet with:
1. Client context summary (what I know about them)
2. Session goals (what we should accomplish)
3. 5 smart questions to ask
4. Potential blockers to watch for
5. Follow-up actions to suggest

## Rules
- Keep it to 1 page — scannable before the call
- Questions should be specific, not generic
- Match my coaching style from brand.md
```

## 11. SOP Creator

**Name:** `my-sop`
**What it does:** Creates a standard operating procedure for any process.

```markdown
---
description: "Create an SOP for any recurring process."
---

# SOP Builder

Ask me: What process do you want to document?

Create an SOP with:
1. Purpose — why this process exists
2. When to use it — trigger/frequency
3. Steps — numbered, specific, clear
4. Tools needed
5. Common mistakes to avoid
6. Quality check — how to know it's done right

## Rules
- Clear enough that someone new could follow it
- No assumed knowledge
- Include specific details, not vague instructions
```
